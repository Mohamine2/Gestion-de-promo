\doxysubsection{Detailed Description}
Implémentation des fonctions de gestion de fichiers pour la promotion d’étudiants. 

Ce fichier contient toutes les fonctions nécessaires pour \+:
\begin{DoxyItemize}
\item lire un fichier texte de données et charger une promotion en mémoire ;
\item sauvegarder et restaurer cette promotion au format binaire ;
\item afficher les informations stockées en mémoire (étudiants, cours, notes, moyennes).
\end{DoxyItemize}
