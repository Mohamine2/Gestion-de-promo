\doxysubsection{Detailed Description}
Définitions des structures de données principales utilisées dans la gestion des étudiants, cours et promotions. 

Ce fichier contient la définition des structures nécessaires à la représentation \+:
\begin{DoxyItemize}
\item des notes d’un cours ({\ttfamily \doxylink{structGrades}{Grades}}),
\item des cours suivis par un étudiant ({\ttfamily \doxylink{structCourse}{Course}}),
\item des étudiants ({\ttfamily \doxylink{structStudent}{Student}}),
\item et d’une promotion entière ({\ttfamily \doxylink{structProm}{Prom}}).
\end{DoxyItemize}

Ces structures servent de base à toutes les fonctions de manipulation des données dans le projet.
