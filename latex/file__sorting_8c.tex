\doxysubsection{Detailed Description}
Implémentation des fonctions permettant de trier et sélectionner les meilleurs étudiants d’une promotion. 

Ce fichier contient l’implémentation des algorithmes utilisés pour \+:
\begin{DoxyItemize}
\item trier les étudiants selon leur moyenne générale ({\ttfamily compare\+Students}, {\ttfamily get\+Top\+Ten\+Students}) ;
\item déterminer les meilleurs étudiants d’un cours donné ({\ttfamily get\+Top\+Three\+Students\+Course}).
\end{DoxyItemize}

Ces fonctions s’appuient sur le tri rapide de la bibliothèque standard ({\ttfamily qsort}) et utilisent des copies de tableaux pour ne pas modifier l’ordre d’origine des étudiants.
