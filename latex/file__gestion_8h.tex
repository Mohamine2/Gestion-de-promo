\doxysubsection{Detailed Description}
Fonctions de gestion de fichiers pour le chargement, la sauvegarde et l\textquotesingle{}affichage d\textquotesingle{}une promotion d\textquotesingle{}étudiants. 

Ce fichier contient les déclarations des fonctions permettant \+:
\begin{DoxyItemize}
\item de lire un fichier texte et de charger une promotion en mémoire ;
\item de sauvegarder une promotion au format binaire ;
\item de restaurer cette promotion depuis un fichier binaire ;
\item d\textquotesingle{}afficher les informations stockées en mémoire.
\end{DoxyItemize}
