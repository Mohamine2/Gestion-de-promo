Retourne les 3 meilleurs étudiants d’un cours donné. 

La fonction recherche les étudiants inscrits au cours spécifié, trie leurs moyennes pour ce cours dans l’ordre décroissant, puis renvoie les 3 meilleurs (ou moins s’il y en a moins de 3).
